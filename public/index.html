Quick Start

Three steps. That's it.

Like here.now for websites, Chain Now abstracts away all blockchain complexity. Agents write JSON; data lands onchain.

Write

POST your data with a chain and key. We handle encoding, signing, and gas.

Track

Read

Read your data back from any chain at any time. Fully verifiable, onchain forever.

Why Chain Now

Skip the blockchain plumbing

Writing onchain traditionally means juggling RPCs, wallets, and gas tokens. Chain Now handles all of it so your agent can focus on what it does best.

Send data to one URL and we route it to the right chain. No endpoints to find or manage.

API Reference

Simple REST API

Four endpoints. No SDK required. Works with any language, any agent, any HTTP client.

POST /api/v1/write

Submit data to write onchain. Send a JSON body with chain (slug), key (string, max 31 chars), and value (any JSON). Returns a txId and statusUrl to track the transaction. Anonymous writes are rate-limited to 5/hr. Include Authorization: Bearer <key> for 60/hr.

GET /api/v1/status/:txId

Check the status of a write transaction. Returns pending, confirmed, or failed, along with the transaction hash and block explorer URL once confirmed.

GET /api/v1/read/:chain/:key

Read data directly from the DataRegistry contract on a specific chain. Returns the decoded value, timestamp, and workflow provenance metadata.

GET /api/v1/chains

List all supported chains with names, chain IDs, and availability status. Add ?testnets=false to exclude testnets.

full-example.sh 
# 1. Write data onchain
TX=$(curl -sS -X POST https://chain-now.vercel.app/api/v1/write \
  -H "Content-Type: application/json" \
  -d '{"chain":"ethereum-sepolia","key":"my-agent","value":{"alive":true}}')

TX_ID=$(echo $TX | jq -r .txId)
echo "Submitted: $TX_ID"

# 2. Poll for confirmation
while true; do
  STATUS=$(curl -sS https://chain-now.vercel.app/api/v1/status/$TX_ID | jq -r .status)
  [ "$STATUS" != "pending" ] && break
  sleep 5
done

# 3. Read it back
curl -sS https://chain-now.vercel.app/api/v1/read/ethereum-sepolia/my-agent | jq .
# → { "value": { "alive": true }, "timestamp": 1711500000, ... }
